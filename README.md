# 🚀 Asteroids

A classic Asteroids game built with HTML5 Canvas — no dependencies, pure vanilla JS.

**[▶ Play it here](https://sdkmode.github.io/asteroids/)**

## Controls

| Action | Keyboard | Touch |
|---|---|---|
| Rotate left | ← / A | Swipe left |
| Rotate right | → / D | Swipe right |
| Thrust | ↑ / W | Swipe up |
| Shoot | Space | Tap |

## Features
- Procedurally shaped asteroids
- Particle explosion effects
- Wrap-around screen edges
- Progressive difficulty (more asteroids per level, faster)
- Touch / mobile support
- Retro CRT glow aesthetic

## Development
Just open `index.html` in any modern browser — no build step needed.
